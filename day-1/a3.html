The above Mini Js Datatype App is work in two phases 1. Creation phase and 2. Execution phase

1 creation phase ::>> it hoisting(taking attendence of Js programme and functions)

2 execution phase ::>> Js will start attaching values to variables and calling functions.

Hoisting::>> Hoisting is taking care of attends of each variable and functions which are present in the programe

Stacks::>> Stacks are data structures that follow the Last-In-First-Out (LIFO) principle, meaning the last item inserted into a stack is the first one to be deleted. In other words, a stack is a list of elements that are accessible only from one end of the list, which is called the Top of Stack

Declaration::>> Creating a variable in JavaScript is called "declaring" a variable.

Reassignment::>> Reassignment consists of assigning a new value for a variable using the = operator.

mutability::>> Mutable is a type of variable that can be changed. In JavaScript, only objects and arrays are mutable, not primitive values.
